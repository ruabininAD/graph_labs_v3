Алгоритм Дейкстры используется для нахождения кратчайшего пути во взвешенном графе с неотрицательными ребрами. Он работает с графом, заданным списком рёбер, и присваивает начальному узлу значение 0, а всем остальным узлам — бесконечность. Далее в каждой итерации алгоритм выбирает узел с наименьшим весом, рассматривает его соседние узлы, изменяет их значения расстояний, если новое значение проще, и помечает текущий узел как посещённый. Алгоритм Дейкстры обычно используется для поиска кратчайшего пути между двумя точками в графе, но можно также использовать его для нахождения кратчайшего пути между одной точкой и всеми остальными. Если в графе есть отрицательные веса, то алгоритм Дейкстры уже может дать неправильный ответ, поэтому для графов с отрицательными весами применяется алгоритм Беллмана-Форда.
