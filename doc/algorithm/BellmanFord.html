Алгоритм Беллмана-Форда используется для нахождения кратчайшего пути во взвешенном графе с отрицательными ребрами. Он работает с графом, заданным списком рёбер, причём не требуется, чтобы граф был связным. Алгоритм присваивает начальному узлу значение 0, а всем остальным узлам — бесконечность. Далее он обходит все рёбра графа и пытается улучшить текущее значение кратчайшего пути до каждого из узлов путём пересчёта расстояний до соседних узлов. При необходимости значения обновляются, пока не будет найден наилучший путь. Если в графе есть цикл отрицательного веса, то алгоритм об этом сообщит.
